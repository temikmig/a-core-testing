# A-Core Test Task

Тестовое задание A-Core: реализована авторизация и страница с деревом объектов с просмотром деталей и фильтрацией.

## Стек

React, TypeScript, GraphQL (Apollo Client), React Hook Form, Feature-Sliced Design (FSD), CSS Modules, UI Kit (`@alphacore/ui-kit`)

## Функционал

- Авторизация
- Дерево объектов (просмотр, раскрытие)
- Фильтрация и поиск
- Просмотр свойств объекта (таблица)

## Примечание

Используется proxy для работы с API (обход CORS)
Проект предназначен для запуска только локально

## Запуск

```bash
npm install
npm run dev
```
